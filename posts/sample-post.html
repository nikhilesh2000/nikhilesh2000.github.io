How to add a new post

To create a new blog post:

  1. Copy this file and rename it (e.g., my-new-post.html)
  2. Update the title in the <title> tag and <h1>
  3. Change the date in the post-meta div
  4. Write your content in the post-content div
  5. Add a preview entry in blog.html
